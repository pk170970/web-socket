# Shared todo app

1. Install Nodejs and VSCode
2. Create a server on backend with file server.ts
3. Install express, socket.io, nodemon
4. Create a public folder which contains index.html, index.js (frontend)
5. Connect client with backend using express.static .... 
6. Initialize websocket on client side using script source as /socket.io/socket.io.js and run io() on javascript file.
